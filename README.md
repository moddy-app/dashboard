## Moddy Dashboard

Official web interface of the **Moddy** bot.

This dashboard allows you to manage and configure Moddy through a modern and intuitive interface, without relying solely on Discord commands.

---

## Tech Stack

* React
* Vite
* TypeScript
* Tailwind CSS
* shadcn/ui (based on Radix UI)

---

## Run the project locally

```bash
cd app
npm install
npm run dev
```
